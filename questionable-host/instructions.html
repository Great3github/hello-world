Installing the file uploader

  1. Install the Tampermonkey browser extension (Chrome, Firefox).
  2. Go to the userscript file: uploader.user.js. Tampermonkey should ask you whether you want to install the userscript, so click “Install.”
  3. Make sure you’re signed into Scratch.
  4. Head to the file uploading page, which only exists with the userscript installed.

If everything went well, you can select files to upload and create links to share with other people.
